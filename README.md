🚀 Mini CRM – Client Lead Management System

A simple and functional Client Lead Management System (Mini CRM) built to help businesses manage leads efficiently.
This project was developed as part of the Future Interns Program.

📌 About the Project
This CRM system allows businesses to:

--> Store leads from contact forms
--> Track lead status (New → Contacted → Converted)
--> Add follow-up notes
--> Manage all leads from a secure admin dashboard

It simulates how real-world companies handle and convert client inquiries.

✨ Features
✔ Lead listing (Name, Email, Source, Status)
✔ Update lead status
✔ Add notes & follow-ups
✔ Secure admin login
✔ Responsive dashboard UI

🔥 Bonus Features 
--> Search & filter leads
--> Timestamp tracking
--> Basic analytics (total leads, conversions)
